# arXiv Submission Checklist

提交前逐项检查，避免被踢回重改。

## 提交前检查

- [ ] **PDF 编译无误**：所有公式、图表、引用正常显示
- [ ] **作者信息完整**：姓名、单位、邮箱、ORCID（如有）
- [ ] **摘要独立**：不含引用、不含公式、不含缩略语（除非通用）
- [ ] **分类正确**：arXiv 分类（如 cs.AI）和 ACM/IEEE 分类号
- [ ] **许可声明**：CC BY / arXiv.org perpetual license
- [ ] **冲突声明**：如有利益冲突需披露
- [ ] **数据/代码链接**：如有补充材料，提供可访问链接
- [ ] **语言润色**：非英语母语建议至少找人校对

## 提交时注意事项

- **标题**：简洁、准确、可搜索
- **摘要**：150-250 词，包含关键词
- **正文**：PDF 必须可检索（不是图片扫描件）
- **引用**：arXiv 优先引用 arXiv 版本（而非会议版本）

## 提交后

- 通常 24-48 小时审核
- 如被拒：按邮件说明修改后重新提交
- 发布后：获得 arXiv ID（如 arXiv:2401.12345）

## 常见错误

- PDF 嵌入字体缺失 → 编译时确保嵌入所有字体
- 分类选错 → 仔细读 arXiv 分类说明
- 重复投稿 → 确保未发表于期刊/会议（预印本允许）
