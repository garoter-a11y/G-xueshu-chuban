# 竞品集成工作流

如何将竞品分析融入 Agent 服务工作流。

## 工作流

1. **识别竞品**：用户提及竞品 → Agent 记录
2. **收集信息**：搜索竞品官网/文档/GitHub
3. **对比分析**：功能/性能/价格/生态对比
4. **输出建议**：基于对比结果给用户建议

## 输出模板

```markdown
## 竞品对比

### A 产品
- 优势：
- 劣势：
- 适用场景：

### B 产品
- 优势：
- 劣势：
- 适用场景：

### 建议
...
```

## 注意事项

- 客观中立，不贬低竞品
- 基于事实，不基于传闻
- 标注信息来源和时间
